# Security Policy

## Supported Versions

| Version | Supported |
| ------- | --------- |
| 0.0.x   | 是（最新版有安全更新） |
| < 0.0.1 | 否（还没发布） |

## Reporting a Vulnerability

请通过 GitHub 的 **Private vulnerability reporting** 功能提交漏洞报告（Security → Report a vulnerability）。

我会在 48 小时内响应，确认收到报告。

- 如果漏洞被确认，我会在 14 天内发布修复版本，并在公开通告中致谢报告者。
- 如果漏洞被我拒绝（不是安全漏洞），我会在报告中说明原因。

感谢你帮助让 `gukasha-rustrade` 更安全！
